#### üì° Sensors (The "Senses")



* Ultrasonic Sensor (HC-SR04): Used for distance measurement and obstacle avoidance.



* LDR (Light Dependent Resistor): Detects light intensity; often used for automatic night lamps or street light projects.



* Sound Detection Sensor: A microphone module used to detect claps or loud noises to trigger actions.



* Infrared (IR) Sensor: Used for line-following robots or detecting nearby objects.



* DHT11 Sensor: Measures both temperature and humidity levels in the environment.



* PIR Motion Sensor: Detects movement within a room, commonly used in security system projects.



* Potentiometer: A variable resistor used as an analog input to control things like LED brightness or servo position.

#### üîå Supporting Tools (The "Foundation")



* Arduino UNO Rev3: The microcontroller "brain" that coordinates all other parts.



* Breadboard: A solderless base used to build and test your circuits quickly.
